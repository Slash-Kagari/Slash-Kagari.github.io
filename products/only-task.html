Only Task

今日やるひとつだけに集中する、シンプルなタスク管理アプリ。

コンセプト

タスク管理アプリは機能を増やすほど、決断を増やし、集中を削ります。Only Task はその逆を目指しました。「今日やるべきひとつ」だけを掲げ、それを終えるまで迷わない。シンプルさの中に、続けるための静かな工夫を入れています。

主な機能

  • 今日のタスクをひとつだけ掲げるホーム画面
  • 完了時の静かな演出（達成感を大切に）
  • 週次/月次のふりかえり
  • 広告非表示などの有料機能（任意）

プラットフォーム

Android（現在クローズドテスト中 / 近日一般公開予定）
